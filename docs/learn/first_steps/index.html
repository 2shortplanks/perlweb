How to get started with your first Perl script

After installing Perl you will want to actually run some Perl code. You need to use a command line (Windows, Mac OSX, Unix/Linux) program to do this.

See if you have Perl installed (ideally a version greater than 5.8, otherwise install a new version): [% WRAPPER code_snippet %]perl -v[% END %]

Download the following file 01_hello_world.pl (maybe into a 'perl_tests' directory - you will need to be able to find this from the command line)
[% WRAPPER code_chunk -%] [%- PROCESS 'first_steps/01_hello_world.pl' -%] [%- END %]

Then from the command line type: [% WRAPPER code_snippet %]perl /path/to/perl_tests/01_hello_world.pl[% END %]

Open the file in a text editor (Windows, Mac OSX, Unix/Linux), and replace NAME, with your name. Save the file (make sure that it stays as plain text, do not save it as RDF or Word).

Run the script again by typing [% WRAPPER code_snippet %]perl /path/to/perl_tests/01_hello_world.pl[% END %]
