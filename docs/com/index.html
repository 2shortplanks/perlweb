What is Perl?

Perl is a highly capable, feature-rich programming language with over [% perl_stats.perl_age %] years of development.

What can Perl do?

Modern Perl runs large and small web sites, CRMs, CMS, streaming media and many other types of systems, oh, and it's still good for sysadmin.
