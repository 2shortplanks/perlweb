After installing Perl you will want to actually run some Perl code. You need to use a command line (Windows, Mac OSX, Unix/Linux) program to do this.

#TODO Video of these instructions being followed out

  • Check your version of Perl: [% WRAPPER code_snippet %]perl -v[% END %]
  • Although you can run code without saving it to a file e.g: [% WRAPPER code_snippet %]perl -e "print qq{hi\n}"[% END %] this introduces more potential issues and it is usually better to save the code to a file...
  • Download the following file 01_hello_world.pl (maybe into a 'perl_tests' directory)
    [% WRAPPER code_snippet -%] [%- PROCESS 'first_steps/01_hello_world.pl' -%] [%- END %]
    Then from the command line type: [% WRAPPER code_snippet %]perl /path/to/perl_tests/01_hello_world.pl[% END %]
