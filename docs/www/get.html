If you are running a Linux system (or most UNIX systems, including Mac OS X), you probably already have an installation of perl. Type perl -v at the command line to find out which version.

Perl Modules from CPAN

The Comprehensive Perl Archive Network (CPAN) is a large repository of perl software, modules, and documentation. On CPAN, you'll find modules organized by author, name, or category. CPAN is mirrored in over [% perl_stats.cpan_mirrors %] locations around the world.
