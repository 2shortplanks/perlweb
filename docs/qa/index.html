Perl Quality Assurance Projects

There are multiple projects in the Perl community related to improving and testing the quality and portability of Perl modules and Perl itself.

Perl-QA

The perl-qa list is the general discussion list for QA of Perl (and modules used to QA Perl code).

Web archive and nntp server

Daily Build

The "smokers" does smoke tests of the bleading edge Perl on various platforms to help the developers spot new bugs as fast as possibly.

Get involved by subscribing to the daily-build list by sending mail to daily-build-subscribe@perl.org (Web archive and nntp server). After subscribing you would probably like to get the Smoke toolkit, see the README for information on that.

CPAN testers

cpan-testers is a group which was initially setup by Graham Barr and Chris Nandor. The objective of the group is to test as many of the distributions on CPAN as possible, on as many platforms as possible.

Subscribe to the cpan-testers mailinglist by sending mail to cpan-testers-subscribe@perl.org. Get more information on how to get involved at http://testers.cpan.org.

Other lists

perl-qa-cvs gets mails with the CVS commits from our sourceforge repository. (Web archive and nntp server).
