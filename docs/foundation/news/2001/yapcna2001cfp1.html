YAPC::America::North's Call for Participation will end on May 1, 2001. Time is running out, so if you are interested in speaking, please visit the CFP web page for more information.

Talks can be:

  • Lightning - 5 minute duration
  • Standard - 20 minute duration
  • Long - 45 minute duration
  • Extra-Long - 90 minute duration
  • Tutorial - 3 hours + break (possibly two sets)
  • Tutorial - Half or full-day
