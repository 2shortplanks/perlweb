The Perl Foundation is pleased to announce five grants approved in September 2003.

  • Daniel Grunblatt - to port the Parrot JIT to IA-64 and HPPA
  • Peter Sergeant - to aid development of RTF parsing modules
  • Autrijus Tang - to develop an I18N framework for Slashcode
  • Leopold Totsch - to rework the Parrot packfile code for multiple code segments
  • Larry Wall - for continuing contributions to Perl

Details can be found here and a press release here.

Many thanks to the individuals and corporations who make the ongoing work of The Perl Foundation possible.
