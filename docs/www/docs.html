Module Documentation

Perl modules come with their own documentation. On UNIX systems, you can access the documentation with the perldoc command.

For example, to get the documentation for IO::File, enter perldoc IO::File from your command prompt.

  • Module documentation is also available at search.cpan.org

The Perl FAQ

Perl FAQs are available as part of your perl distribution with the
perldoc perlfaq command.

  • Read FAQs online at faq.perl.org
