A good setup for Perl on Windows (32 and 64 bit):

  • Window tools
  • Make sure you do not have a Perl already installed [%# (including from msys - but people might need for some other build systems %] - uninstall it if you do (see video below).
  • Download and install Strawberry Perl Professional (Perl version [% strawberry_pro_perl_version %]).
  • Go to your start menu, then click the "Perl command" link.
  • Install App-cpanminus by typing in: [% WRAPPER code_snippet %]cpan App::cpanminus[% END %]

Explanation of this setup

  • Strawberry Perl Professional is recommended, even though it is technically in beta and not the latest version of Perl. The big advantage is that many useful Perl modules (especially those tricky to install on Windows) are part of the installation.
  • To interact with the command line and run Perl commands, you need to run 'cmd'. There are better alternatives available as well.
  • App-cpanminus provides the cpanm tool, which makes installing modules very simple to do.
