Parrot 0.4.8 "Eponymous" Released!

On November 14th, 2006, Parrot pumpking Jerry Gay released Parrot 0.4.8. See the Parrot 0.4.8 release announcement for more details or download Parrot 0.4.8 from the CPAN.

What is Parrot?

Parrot is a virtual machine designed to efficiently compile and execute bytecode for dynamic languages. Parrot currently hosts a variety of language implementations in various stages of completion, including Tcl, Javascript, Ruby, Lua, Scheme, PHP, Python, Perl 6, APL, and a .NET bytecode translator.

Parrot is not about parrots.

How to Get Involved

On Saturday, 13JAN07, please join us on IRC in #parrot (irc.perl.org) for Parrot Bug Day to work on closing out as many RT tickets as possible for parrot.

Parrot development normally takes place on the parrot-porters mailing list. Subscribe by sending mail to parrot-porters-subscribe@perl.org or read the NNTP archive.

There is documentation in the docs subdirectory of the Parrot source, with an snapshot here of svn-latest documentation.

There is a list of TODO items, as well as notes on how to keep parrot's cage clean.

Patches are welcome. See the patch FAQ for details.

View and manipulate bugs and TODO items at the Parrot bugs page.
