[% page.title ="Who's Who in Perl6" %] Architecture Team
  • Damian Conway is helping Larry Wall with the design of Perl 6, providing a different perspective from most. He also writes the Exegeses, which explain by example Larry's design documents (the Apocalypses).
  • Larry Wall is the inventor of Perl, rn, patch, and many other wonderful things.  He is the language designer for perl6.
  • Dan Sugalski is the internals chief for perl6.  His job is to coordinate the design of the parser, compiler, optimizer, and runtime engine.
  • Nathan Torkington is the project manager.  His job is to herd cats.
Parrot Code Manager / Pumpking
  • Leopold Toetsch is currently the leader of the Parrot development team.
  • Steve Fink was the code cheif for Parrot 0.0.9-0.0.12.
  • Jeff Goff was the code chief for Parrot 0.0.5- 0.0.8.
  • Simon Cozens was the the code chief for Parrot 0.0.1-0.0.4.
Resident Journalists
  • Piers Cawley writes the "This Week on Perl 6" summaries, and has been since June 2002.
  • Bryan Warnock used to compile the Perl 6 Digest with Simon Cozens.
