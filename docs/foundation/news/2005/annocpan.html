[% page.banner = "March 1, 2005 - Grant Given to Ivan Tubert-Brohman for AnnoCPAN" %] [% page.title = page.banner %]

The Perl Foundation is proud to announce the latest grant aimed at furthering development in Perl, Ivan Tubert-Brohman will receive $1000 as a development grant to implement AnnoCPAN, a web interface for the documentation of all the modules on CPAN, where users can add annotations on the margin of specific paragraphs throughout the POD.
