Learning perl

Resources and references for learning Perl

So, you want to learn perl? A good place is this introduction or any of the books we recommend.

You can read the perl documentation on our perldoc site. Did you know that you can also access the perl documentation with the perldoc command? Try "perldoc perlreftut" at your command line prompt.

Mailing Lists

There are many Perl related mailing lists. An exhaustive list is available at lists.perl.org.

If you're new to Perl, consider joining the beginners list.

You can read the beginners lists (and all other perl.org mailinglists) with your newsreader (Unix, Windows or MacOS readers) from the news server nntp.perl.org. You can also browse it on the web
