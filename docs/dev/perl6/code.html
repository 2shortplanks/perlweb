What's Parrot

Parrot is a virtual machine used to efficiently execute bytecode for interpreted languages - specifically, Perl 6, although we would like to keep the door open for other languages in the future. As such, Parrot will be the core of the Perl interpreter; it will be the target platform to which Perl 6 code is compiled.

Where are we

Parrot source code was released to the world on Monday 10th of September 2001; since then, an army of developers have furiously been hacking away on it. Currently, we can assemble the Parrot assembly language to bytecode and execute it with a test interpreter.

The Perl scalar type (string, integer and floating-point) has been implemented, along with basic array and hash types. We have global symbol tables, subroutines, a JIT, and a good intermediate language compiler.

Sean O'Rourke has contributed a working Perl 6 parser and compiler for a significant subset of the perl 6 definition that currently exists.

How to get the code

Periodic releases will appear on CPAN; the current release is version 0.0.10, and can be found in the CPAN source directory.

For the very latest version of Parrot, check out the source from CVS. There is anonymous access to all, and commit access to those who have taken responsibility for various areas of the code or produced a large number of high-quality patches.

Frequent automatic snapshots of the CVS tree are available here.

For automatic smoke testing you can use rsync to synchronize a directory with a checked-out copy of the latest CVS tree.

    rsync -av --delete cvs.perl.org::parrot-HEAD parrot

How to get involved

Parrot development takes place on the perl6-internals mailing list. If you follow this list, you'll be able to get an idea of what sort of things are needed. Subscribe by sending mail to perl6-internals-subscribe@perl.org or read the web archive or the nntp archive. When development of the Perl6 language needs an internals list we will move the parrot list to the parrotcode.org domain.

There is documentation in the docs subdirectory of the Parrot source, and the design documents can be found here.

A TODO list is being developed, and can be found here.

Patches of all natures are welcome. See the patch FAQ for details.

Bugs and to-do items can be viewed and manipulated at the Parrot bugs page.
