Welcome to learn.perl.org

Helping you get started with Perl 5 (What is Perl?)....

This site will take you through installing Perl 5, your first program and some basic best practices.

We'll also introduce you to Perl Modules, a few examples, the Perl documentation and Perl books (some freely available online) as well as other resources.

Perl offers many advantage over other languages, including CPAN which has [% perl_modules | comma %] modules ready to help you with many different types of task.

Finally we'll show you where to get more help and how to join the Perl community - one of the best aspects of Perl.

Existing developers / sysadmins

If you are already a developer or systems administrator, in any language, on any platform, please skim through the site, there are lots of details and resources which you will still find useful. It is also worth reviewing to find out what 'Modern' Perl actually is.
