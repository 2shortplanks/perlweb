Where will I find an answer?

If you are just getting started, you probably have lots of questions, here are some places to start.

Perl FAQ

You can see the latest Perl Frequently Asked Questions on this site, all versions of Perl also come with perldoc perlfaq).

Documentation

You can read the Perl documentation on our perldoc site. You can also access the Perl documentation with the perldoc command. Try "perldoc perlreftut" at your command line prompt.

Mailing Lists

You can get help on the dedicated Perl beginners mailing list.

Community

One of Perl's big strengths is it's community. See if you have a local "Perl Mongers" group, or if there are any events happening near you.

IRC

IRC (Internet Relay Chat) is a great place to get help if you're polite and ask smart questions.

  • irc.perl.org
